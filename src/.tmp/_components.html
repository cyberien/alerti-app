Input group merge

.input-control-merge should be added to the .input-group container to enable input group text merge with form controls. .form-control should always go first and followed by .input-group-prepend or .input-group-append. .form-control-prepended or .form-control-appended should be added to the .form-control input to ensure proper paddings and border radius values.
